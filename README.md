# Notes App Frontend

Frontend for the Notes Application built using HTML, CSS, and JavaScript.

## Features

* Create Notes
* View Notes
* Delete Notes
* Responsive User Interface
* Connected with Render Backend API

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Fetch API

## Installation

Clone the repository:

```bash
git clone https://github.com/SkMusa786/notes-frontend.git
cd notes-frontend
```

Open:

```bash
index.html
```

or use Live Server.

## Backend API

The frontend communicates with:

```
https://notes-backend-zelh.onrender.com/api/notes
```
